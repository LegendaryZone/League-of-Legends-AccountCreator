# League-of-Legends-AccountCreator

~~~~~~Created by Andreas Giannoulis~~~~~~
The app is targetted to create league of legends accounts for new users or people who bot. The script currently supports EUNE/EUW/NA server
but will be soon update to include the rest of them. The script is not using any illegal methods or proxies.

REQUIREMENTS:
1) Selenium
2) ChromeWebDriver
3) 2Captcha Subscription [I will try to add more services soon, 2captcha was the first to pop up :)]
4) A decent connection if you want faster creation!
